複数のカーソル

MacであればCmd（command）キーを、Windows/LinuxであればCtrlキーを押しながら、タイピングしたいすべての場所をクリックすると、複数の個所に同一のテキストを入力できる。

行の複製

Cmd + Shift + D (Mac)
Ctrl + Shift + D (Windows/Linux)

現在選択中の行の上または下へ移動

Cmd + Ctrl + ↑または↓ (Mac)
Ctrl + ↑または↓ (Windows/Linux)

選択中の文字の第二候補の選択

Cmd + D (Mac)
Ctrl + D (Windows/Linux)
これで、強調表示された言葉や、第二候補の言葉または文字を選択できる。

選択中の文字の第二候補の選択解除

Cmd + U (Mac)
Ctrl + U (Windows/Linux)
このキーボードショットカットで直前に選択した文字を逆順で解除できる。

コメントのオン・オフ切り替え

Cmd + / (Mac)
Ctrl + / (Windows/Linux)
